## 🚀 Featured Projects

### 🔁 [Zagazig Branch ELT Sales Pipeline](https://github.com/samawael7)
`SQL Server` `Apache Airflow` `dbt` `Docker` `Medallion Architecture`
> End-to-end ELT pipeline ingesting CRM & ERP data into a SQL Server warehouse using Bronze/Silver/Gold layers. Orchestrated with Airflow, transformed with dbt across fact and dimension tables with incremental loads.

---

### 📚 [Gravity Books – Data Warehouse & ETL Pipeline](https://github.com/samawael7)
`SQL Server` `SSIS` `Star Schema` `Power BI` `DAX`
> Star Schema DWH with 6 dimensions and 1 fact table processing 15K+ sales transactions. SSIS ETL workflows migrating OLTP data to dimensional model. Power BI dashboard analyzing 11K+ books and 2K+ customers.

---

### 📊 [User Analytics Dashboard](https://github.com/samawael7)
`Python` `Pandas` `Seaborn` `Matplotlib`
> End-to-end pipeline fetching from a public API, parsing nested JSON, cleaning records, and visualizing user demographics and distribution insights.

---

### 🧴 [Velora – Skin Care & Disease Detection](https://github.com/samawael7)
`Python` `CNN` `SQL` `APIs` — *Graduation Project — A+*
> ML pipeline classifying 7 skin conditions from 10K+ medical images (HAM10000) with ~77% CNN accuracy. Led front-end workflow development and contributed to the full ML pipeline.
